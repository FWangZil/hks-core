# 守护精灵

## 已完成功能

### 用户
* 用户登录
* 用户注册
* 用户退出登录
* 用户报警
* 记录用户位置
* 提供用户运动轨迹点
* 用户绑定亲友功能

### 报警
* 用户报警
* 更新案件信息
* 列出历史报警信息
* 获取报警信息详情


## 构建情况
* 在服务器上通过docker-compose 构建了 mysql redis 与本服务
* 通过端口映射到80端口对web端与Android端提供了接口
